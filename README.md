# Virtual-kiosk
Virtual-kiosk is a lightweight linux addressed interface, this project is aimed at an easier interaction for users with a set of predefined Virtual Machines.

The project focuses on an homogenuous simple and straight forward controlled launcher of vm's. Halting non technical personel destructive accidents and keeping the engine oracle virtualbox with a layer of protection from unintended user modifications.

No external resources will be provided, such as machine images, disk files, or proprietary resources.

---
# Motivation
The origin came to be from the need to operate legacy systems through virtualization, avoid legacy hardware failure and its obsolecency.

---
# The goals 
Are to simplify the acces of a series of oracle virtualbox vm's, with their series of presets.
Ease the hurdle of less technical personal to operate the vm's.
Avoid user errors and accidental modifications by providing a layer to lock settings of the Virtual enviroments.

---
# Compatibility
This project addresses linux specifically ubuntu based linux mint 22.3.
